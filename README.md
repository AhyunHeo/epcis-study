# epcis-study

## To run epcis 2.0 on docker
```console
cd epcisV2
docker-compose pull
docker-compose up
```

## EPCIS 2.0 REST API Documentation
```console
cd src
npm install
node openApi.js
```
Go to browser and access localhost:8081/api-docs
